## 📝 プロジェクト紹介

翰林大学の学生がより便利に学祭に参加できるよう、また安全な学祭運営を支援するために開発されたウェブサイトです。  
さらに、**居酒屋予約システム**を翰林大学で初めて導入し、安全性を強化するとともに、  
さまざまな情報を効率的に提供するプラットフォームを構築しました。
